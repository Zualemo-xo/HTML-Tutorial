Colours in html

Plain black would be the text color if not for CSS! Dosen't that feel bland? Well no worries,as wea are gonna be using colours to lighten things up!

The style tags used for this purpose are :

  1. style="color: #0000FF;"---To specify color using Hexadecimal values
  2. style="color: rgb(0,0,255);"---Red Green and Blue values lie from 0 to 256
  3. style="color: red;"---If you're lazy to search up em specific codes or values,just use the color name.

On a side note a fade effect is possible by setting the 4th coordinare in rgba with values between 0 and 1.

Inline CSS[Styling inside the tags themselves] was used to assign colours to the text used here.

Physical styles
Physical styles are used to indicate direct changes to text physically. The tags include:
<b>
Bold tag used to change the font pattern.
<big>
Increases the font size by one when use everytime.
<i>
Used to provide an italic pattern.
<tt>
A typewriter font is obtained(Similar to Courier New)
<small>
Decreases the font size by 1 each time used.
<sub>
Used to obtain a subscript.
<sup>
Used to obtain a superscript.

Tabulations and Calculations

We finally turn our attention towards tables(the box kind).HTML has a simple way to create these.

TagDescription
<table> This tag is a container for every other tag used to create a table.
<tr> It stands for table row.The opening and closing tr tags surround the cells for that row
<th> th stands for table header.Used when defining the header information.
<td> td stands for table data and contains the content for the cell.
